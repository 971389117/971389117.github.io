流程控制

if

if 方括号中【num2*2-num1+20-20】计算结果为100，结果输出两个数相等
在方括号中，可以进行算术运算

num1=100
num2=100

if test $num1 -eq $[num2*2-num1+20-20]
then 
    echo '两个数相等'
else
    echo '两个数不相等'
fi

文件权限

递归的更改文件属组：chgrp [-R] 属组名 文件名
更改文件属主，以及属组
chown [-R] 属主名 文件名
chown [-R] 属主名:属组名 文件名

更改文件属性
数字法：
r = 4 w = 2 x = 1
7 可读可写可执行
6 可读可写
5 可读可执行
4 可读
3 可写可执行
2 可写
1 可执行
0 不可读不可写不可执行

chmod [-R] 777 文件或目录

字母法：
u = user g = group o = other a = all

直接赋值：chmod u=rwx,g=rx,o=r 文件名
增量赋值: chmod u+x 文件名
拿掉所有人可执行权限: chmod a-x 文件名
